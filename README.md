# ！！百度云不限速下载！！

**Motrix扩展版，最新百度云SVIP加速，新建任务选择百度云加速即可。**

## 安装教程

Windows下载**Motrix Setup x.x.xx.exe**即可(版本不固定)，MacOS下载**Motrix-x.x.xx.dmg**(版本不固定).

## 使用说明

点击Motrix添加任务的按钮，切到百度云页面可以解析百度云文件，实现SVIP加速，我们挑战的是**跑
满带宽**，而不是不限速的1Mb/s-10Mb/s，追求的就是**跑满带宽**.
